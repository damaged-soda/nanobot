# Agent Entry Points

This repository keeps the upstream nanobot product documentation and a separate long-term new-kernel roadmap.

For normal nanobot development, follow the repository's existing code, tests, and documentation patterns.

For tasks that explicitly touch the new event-sourced runtime kernel roadmap, architecture notes, ADRs, or plans, read and follow `docs/new-kernel/AGENTS.md` before making changes. Those rules are scoped to the new-kernel effort and should not be applied globally unless the task says so.
